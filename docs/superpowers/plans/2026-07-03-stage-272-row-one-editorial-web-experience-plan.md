# Stage 272 ROW ONE Editorial Web Experience Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Improve the generated ROW ONE website so the existing `row-one-app/v2` organization appears as a professional editorial fashion-intelligence experience in static HTML.

**Architecture:** Keep ROW ONE as a local static site renderer. Rework `templates.py` to add richer homepage editorial rails, article detail contents, stronger evidence cards, and refined CSS/JS hooks using existing `RowOneEdition` and `RowOneStory` fields only; keep app, manifest, and runtime schema contracts unchanged.

**Tech Stack:** Python 3.11+, static HTML/CSS/vanilla JS generated by `templates.py`, pytest, ruff, uv.
